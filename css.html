What is CSS?

CSS stands for Cascading Style Sheets.

CSS is the language we use to style a Web page.

External stylesheets are stored in CSS files.

Why to learn CSS?

CSS describes how HTML elements are to be displayed on screen, paper, or in other media.

If Html is body of the web page then css is the external design i.e, skin and looks of the web page.

CSS saves a lot of work. It can control the layout of multiple web pages all at once.

It allows one to adapt the presentation to different types of devices, such as large screens, small screens, or printers.

Sometimes, CSS is independent of HTML and can be used with any XML-based markup language.
